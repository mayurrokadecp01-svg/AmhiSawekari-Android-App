AmhiSevikari Android

A devotional Android application built using Jetpack Compose, Clean Architecture, Firebase Firestore, and modern Android development practices.

Overview

AmhiSevikari is a devotional application developed for Dagdusheth Datta Mandir, Pune.

The application enables devotees to browse categorized bhajans, access devotional lyrics, and synchronize content from Firebase Firestore while supporting modern Android architecture principles.

Features

Devotional Content

* Browse Bhajan Categories
* View Bhajan Collections
* Read Bhajan Lyrics
* Fast Search Experience

Synchronization

* Firebase Firestore Integration
* Content Synchronization
* Offline Data Storage
* Background Sync Support

User Experience

* Jetpack Compose UI
* Responsive Navigation
* Material Design Components
